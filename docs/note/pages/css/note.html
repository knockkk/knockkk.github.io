# CSS基础

# 选择器优先级

  1. 内联样式（如<h1 style="color:red">Hello~</h1>）
  2. id选择器 （如#example）
  3. 类选择器 （如.example）
  4. 类型选择器 （如div）
  5. * 通配符选择器

important

当在一个样式声明中使用一个 !important 规则时，此声明将覆盖任何其他声明。
